## 🌌 About Quantex

**Quantex** is a next-generation, open-source code editor designed for developers who value simplicity, speed, and intelligence.  
Built using **Python + PyQt6 + QScintilla**, it offers a beautiful, customizable interface and essential development features out of the box.

Whether you're editing scripts or building large applications, Quantex gives you the tools and performance to do it faster — with style.

---

## ✨ Key Features

- 🧠 **Smart Syntax Highlighting** — Full language-aware highlighting using QScintilla.
- ⚡ **Auto-completion Engine** — Get real-time suggestions as you type.
- 💬 **Tooltip on Hover** — Instantly view function definitions and documentation.
- 🧩 **Plugin-Ready Architecture** — Extend Quantex easily with community-built plugins.
- 🖤 **Modern UI** — Clean, distraction-free design using PyQt6 widgets.
- 🌈 **Cross-Platform** — Works flawlessly on Windows, macOS, and Linux.
- 🚀 **Lightweight & Fast** — Built for speed without unnecessary bloat.
